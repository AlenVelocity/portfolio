## Install & run

Make sure you have nodejs and npm installed. Install dependencies with:

```bash
npm
```

Once it's done start up a local server with:

```bash
npm start
```

To view the components storybook:

```bash
npm run storybook
```

To create a production build:

```bash
npm run build
```
